MoreAgent — 自进化代码质量引擎

CLI 入口 — moreagent.py

  • • 子命令路由: scan / fix / status / setup / cron / report / history
  • • .env 自动加载 (DEEPSEEK_API_KEY 等)
  • • sys.path 管理: src/ 和项目根自动加入
  • • Windows兼容: encoding="utf-8" 避免 GBK 错误

协调者 — 分层委托决策

  • • should_delegate(): 复杂度/成功率/预算三维决策
  • • Layer 1: token < 1000 自主执行 (Enterprise Fixer)
  • • Layer 2: 委托给 Analyzer/Coder/Tester/Reviewer
  • • 169个 agency-* 角色技能 · 能力地图动态评估

核心流程 — 扫描→修复→门控→验证→学习

  • • ① 扫描: Tiered(规则→Ollama→API) × 8维AST
  • • ② 修复: 5类修复器 (enterprise/llm/experience/dead/unused)
  • • ③ 门控: ≥0.8自动 · 0.5~0.8待审 · <0.5拒绝
  • • ④ 验证: 语法+逐修复回归+测试+import链
  • • ⑤ 学习: 经验积累 → 置信度校准 → 成功3次→新Skill

知识层 — 经验闭环自进化

  • • FAISS向量索引: 修复经验向量化存储
  • • Experience Retriever: 相似问题→历史方案自动注入
  • • 置信度校准: 根据历史成功率动态调整修复器置信度
  • • 经验传播: 校准结果自动同步到修复器运行时配置

分级扫描 — 三级降级策略

  • • Level 0: 纯规则 (AST+正则) → 零成本，覆盖60%
  • • Level 1: 本地Ollama小模型 → 几乎免费，覆盖80%
  • • Level 2: DeepSeek API → 按token计费，覆盖95%
  • • 启动自动探测最高可用级别，失败自动降级

基础设施 — 生产级保障
